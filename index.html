Segurança

  • Cada Access Point possui uma senha de Wi-Fi distinta.
  • Credenciais de acesso aos roteadores e switches não são as de fábrica.
  • Ausência de uma rede para visitantes, forçando o compartilhamento da senha interna.

Falhas Encontradas

  • Cabos de rede sem padrão (CAT5, CAT5e, não especificados).
  • Cabeamento de rede passado junto à fiação elétrica sem blindagem.
  • Roteadores mal configurados, causando quedas de performance.
  • Access Points mal distribuídos ou subutilizados.

Plano de Ação e Recomendações

Imediatas (Sem Custo)
Avançadas (Com Investimento)
  • Criar Rede de Visitantes: Isolar o tráfego de convidados para garantir performance e segurança da rede interna.
  • Realocar AP "posto central 2": Mover para a ala da pediatria, otimizando a cobertura e eliminando redundância com o AP "Corredor".
  • Desativar Wi-Fi Direct das Impressoras: Reduzir a interferência nos canais de 2.4GHz, já que as impressões são feitas via rede cabeada.
  • Reconfigurar APs Inativos/Lentos: Padronizar senhas e configurações dos APs do 2º andar para torná-los utilizáveis e eficientes.
  • Padronizar Nomes de Rede (SSID): Renomear todas as redes para um padrão claro, como PostoCentral_Setor (ex: PostoCentral_Recepcao, PostoCentral_Vacinas).
  • Organizar Sala de Servidores: Limpar e refazer a identificação dos cabos com etiquetas duráveis para facilitar a manutenção futura.
  • Reconfigurar os roteadores: Visando reduzir a sobreposição de canais e consequentemente a interferência, é recomendado utilizar a largura de 20MHz na faixa de 2.4GHz aumentando a qualidade da conexão em lugares com alta densidade de dispositivos.
  • Manter ou configurar roteadores 5GHz: Configurando para operar com 40MHz ou 60MHz é possível garantir uma maior largura de banda para aplicações que demandem mais velocidade.
  • Implementar Rede Mesh: Adquirir um sistema Mesh com pelo menos um roteador adicional para criar uma cobertura unificada e estável em todo o posto.
  • Upgrade Geral da Infraestrutura: Para suportar velocidades maiores, é necessária a troca de switches, access points e todo o cabeamento para um padrão superior (CAT6 ou superior).
  • Utilizar Cabos Blindados (STP): Substituir o cabeamento que passa junto à rede elétrica por cabos blindados para eliminar interferência eletromagnética e garantir a integridade do sinal.
